# ایران شگفت‌انگیز

مجموعه‌ای از پروژه‌ها درباره ایران، فرهنگ، ادبیات و زبان پارسی، تاریخ و بیشتر.

English version: [Readme.en.md](Readme.en.md)

## محیط زیست و طبیعت

- [wildfire MCP Server](https://github.com/aliafsahnoudeh/wildfire-mcp-server): یک ارائه‌دهنده مدل کانتکس برای تشخیص آتش‌سوزی در ایران.

## زبان و ادبیات پارسی

### مخازن گیت‌هاب

- [Shahnameh Dataset](https://github.com/aliafsahnoudeh/shahnameh-dataset): نسخه کامل شاهنامه در قالب‌های مختلف مانند PostgreSQL، JSON و SQLite
- [Shahnameh API](https://github.com/ArezooGoshtasbi/shahnameh-api): یک API بک‌اند مبتنی بر FastAPI پایتون با اندپوینت‌های کاربردی برای جستجو و پرس‌وجوی فصل‌ها، ابیات و حتی توضیحات شاهنامه.
- [Shahnameh MCP Server](https://github.com/aliafsahnoudeh/shahnameh-mcp-server): ارائه‌دهنده مدل کانتکس برای شاهنامه.
- [JumpLander-Persian-Forum-Dataset](https://github.com/ratitya/JumpLander-Persian-Forum-Dataset): دسترسی به یک مجموعه داده ساختاریافته از فروم‌های فارسی برای بهبود مدل‌های NLP در طبقه‌بندی متن، خلاصه‌سازی و جستجوی معنایی به زبان فارسی.

### برنامه‌های وب

- [Shahnameh Doulingo Web](https://shahname.vercel.app/): خوانش و یادگیری شاهنامه به سبک دولینگو.

### برنامه‌های موبایل

- [Ferdowsi's poems for Android](https://play.google.com/store/apps/details?id=air.com.arsnetworks.poems.ferdosi): نرم‌افزاری شامل مجموعه‌ای از اشعار فردوسی که با تلاش فراوان گردآوری شده‌اند.

## جغرافیا و نقشه‌ها

- [Is in Iran](https://github.com/aliafsahnoudeh/is-in-iran): یک کتابخانه پایتون سبک است که بررسی می‌کند آیا یک نقطه جغرافیایی (عرض جغرافیایی، طول جغرافیایی) در داخل مرزهای ایران قرار دارد یا خیر.
- [react-iran-maps](https://github.com/pq1234s/react-iran-maps): ایجاد نقشه‌های تعاملی SVG ری‌اکت از ایران با ویژگی‌های دریل‌داون، با استفاده از d3-geo و topojson برای نمایش دقیق استان‌ها و شهرستان‌ها.

## گاه‌شمار و زمان

- [persian-gnome-calendar-extension](https://github.com/IamRezaMousavi/persian-gnome-calendar-extension): نسخه‌ای فورک شده از افزونه گنوم کلندر با طعم فارسی
- [irancal](https://github.com/masoudd/irancal): تقویم ایرانی در ویندوز

## حمل و نقل

- [mashhad-metro] (https://github.com/shahriaarrr/mashhad-metro): یک اپ موبایل برای راهیابی در مترو شهر مشهد

## زبان برنامه‌نویسی

- [SalamLang] (https://github.com/SalamLang/Salam): زبان سلام: اولین زبان برنامه‌نویسی برای فارسی‌زبانان و عرب‌زبانان، الهام گرفته از کلمه سلام به معنای صلح. با رویکردی کاربرپسند، تجربه برنامه‌نویسی قابل دسترسی را فراهم می‌کند و همکاری و سادگی را برای توسعه‌دهندگان محلی ترویج می‌دهد.

## مالی

- [pricedb](https://github.com/margani/pricedb): پایگاه داده قیمت به‌روزرسانی خودکار برای نرخ‌های ارز ریال ایران (IRR) در بازار آزاد ایران

## اعتبارسنجی داده‌ها

- [zod-ir](https://github.com/Reza-kh80/zod-ir): ابزار رسمی Zod برای اعتبارسنجی داده‌های ایرانی و استخراج فراداده‌ها: کد ملی، شماره تلفن، کارت بانکی، کد پستی و غیره.
