# Awesome Iran

A collection of awesome projects about Iran, its culture, literature and Persian language, history, and more.

## Geography and Maps

- [Is in Iran](https://github.com/aliafsahnoudeh/is-in-iran): A lightweight Python library that checks whether a given geographic poin (latitude, longitude) lies inside the borders of Iran.

## The Persian Language and Literature

### Github Repositories

- [Shahnameh Dataset](https://github.com/aliafsahnoudeh/shahnameh-dataset): Full version of Shahnameh in different formats like PostgreSQL, JSON and SQLite
- [Shahnameh API](https://github.com/ArezooGoshtasbi/shahnameh-api): A Python FastAPI based backend API with useful endpoints to search and query Shahnameh's chapters, verses and even explanations.
- [Shahnameh MCP Server](https://github.com/aliafsahnoudeh/shahnameh-mcp-server): A Model Context Provider for Shahnameh.

### Web Applications

- [Shahnameh Doulingo Web](https://shahname.vercel.app/): Reading and learning about Shahnameh in Doulingo style.

### Mobile Apps

- [Ferdowsi's poems for Android](https://play.google.com/store/apps/details?id=air.com.arsnetworks.poems.ferdosi): Ferdowsi software is a collection of poems that have been collected with great effort.
