# Awesome Iran

A collection of projects about Iran — its culture, literature, Persian language, history, and more.

نسخه فارسی: [Readme.fa.md](Readme.fa.md)

## Nature and Environment

- [wildfire MCP Server](https://github.com/aliafsahnoudeh/wildfire-mcp-server): A Model Context Provider for Wildfire Detection focused on Iran.

## The Persian Language and Literature

### Github Repositories

- [Shahnameh Dataset](https://github.com/aliafsahnoudeh/shahnameh-dataset): Full version of Shahnameh in different formats like PostgreSQL, JSON and SQLite
- [Shahnameh API](https://github.com/ArezooGoshtasbi/shahnameh-api): A Python FastAPI based backend API with useful endpoints to search and query Shahnameh's chapters, verses and even explanations.
- [Shahnameh MCP Server](https://github.com/aliafsahnoudeh/shahnameh-mcp-server): A Model Context Provider for Shahnameh.
- [JumpLander-Persian-Forum-Dataset](https://github.com/ratitya/JumpLander-Persian-Forum-Dataset): Access a structured Persian forum dataset to enhance NLP models for text classification, summarization, and semantic search in Farsi.

### Web Applications

- [Shahnameh Doulingo Web](https://shahname.vercel.app/): Reading and learning about Shahnameh in Doulingo style.

### Mobile Apps

- [Ferdowsi's poems for Android](https://play.google.com/store/apps/details?id=air.com.arsnetworks.poems.ferdosi): Ferdowsi software is a collection of poems that have been collected with great effort.

## Geography and Maps

- [Is in Iran](https://github.com/aliafsahnoudeh/is-in-iran): A lightweight Python library that checks whether a given geographic poin (latitude, longitude) lies inside the borders of Iran.
- [react-iran-maps](https://github.com/pq1234s/react-iran-maps): Create interactive React SVG maps of Iran with drill-down features, using d3-geo and topojson for detailed province and county views.

## Calendar and Time

- [persian-gnome-calendar-extension](https://github.com/IamRezaMousavi/persian-gnome-calendar-extension): A forked version of Gnome Calendar extension with Persian taste
- [irancal](https://github.com/masoudd/irancal): Iranian Solar Hijri Calendar for windows

## Transportation

- [mashhad-metro](https://github.com/shahriaarrr/mashhad-metro): A complete mobile app for navigating Mashhad Metro system.

## Programming Language

- [SalamLang](https://github.com/SalamLang/Salam): Salam Language: The inaugural coding language for Persian and Arabic speakers, inspired by the word salam meaning peace. With a user-friendly approach, it provides an accessible coding experience, promoting collaboration and simplicity for local developers.

## Finance

- [pricedb](https://github.com/margani/pricedb): Auto-updated pricedb for Iranian Rial (IRR) exchange rates in Iran's free market

## Data Validation

- [zod-ir](https://github.com/Reza-kh80/zod-ir): Official Zod utility for Iranian data validation & metadata extraction: national code, phone number, bank card, postal code, etc.
